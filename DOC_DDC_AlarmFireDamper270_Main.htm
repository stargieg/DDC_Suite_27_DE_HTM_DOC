Brandschutzklappe @@&name@@

 

Ansteuerung und Überwachung einer motorisch betriebenen Brandschutzklappe. (siehe auch „Motorische Brandschutzklappen, Gruppe @@&ref@@“)

 

Die übergeordnete Gruppenfunktion kann einen Klappentest herbeiführen, bei dem die Klappe erst geschlossen, dann geöffnet wird. Die Rückmeldungen werden überwacht, das Ergebnis des Tests wird in der Gruppenfunktion abgebildet.

 

Die Laufzeiten werden aufgezeichnet, so dass rechtzeitig erkennbar wird, ob ein Motor ggf. ausgetauscht werden muss.

 

Die Meldungen <Rückmeldung Auf fehlt> und <Rückmeldung Zu fehlt> werden einzeln zur weiteren Verarbeitung ausgegeben. Zusätzlich kann ausgewählt werden ob eine Sammelmeldung als Störung oder Meldung bzw. keiner oder beide im Programm weiterverarbeitet werden soll.

 

Parameter

Wert

Virtueller Eingriff (0=Zu, 1/2=Auto, 3=Auf)

@adj_Vorwahl.0p@

Rückmeldefehler sind Quittierungspflichtig (0=Nein, 1=Ja)

@adj_QuitTyp.0p@

Sammelalarmtyp (0=keiner, 1=Meldung, 2=Störung, 3=beides)

@adj_SmTyp.0p@ °C

Zugehörige Gruppe
